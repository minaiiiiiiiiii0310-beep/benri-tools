💑 配偶者控除・配偶者特別控除 判定 シミュレーター

本人の 年収 + 配偶者の 年収 を 入れるだけで、配偶者控除 / 配偶者特別控除 が 適用できるか、控除額 は いくらか、節税額 は いくらか を 自動 計算。103万・150万・201万 の 壁 も 一目で 判定。

配偶者控除・配偶者特別控除 は、配偶者の 年間給与収入が 一定額 以下の とき、本人の 所得税・住民税から 一定額を 差し引ける 制度。専業主婦 (主夫) 世帯や パート共働き 世帯では 家計に 直結する 重要な 節税 ポイントです。配偶者の 年収が 103万円 以下 なら 「配偶者控除」 (本人 年収 1095万以下で 所得税 38万・住民税 33万)、103〜201万円 なら 「配偶者特別控除」 が 段階的に 適用され、150万円を 超えると 急速に 控除額が 減少する ため 150万円の 壁 と 呼ばれます。さらに 106万 / 130万円の 社会保険の 壁、103万円の 所得税の 壁 と 各種の 壁が 入り乱れて 「働き損」 が 心配される 領域でも あります。本ツールでは 本人・配偶者の 年収と 年齢を 入力するだけで、適用区分・控除額・節税額を 即時 算出し、世帯全体で 手取りが 最大化する 働き方の 検討に 役立ちます。

配偶者控除・特別控除 とは — 適用要件

配偶者の 所得が 一定以下 のとき、本人 の 所得から 一定額を 控除できる 制度。所得税・住民税 が 安くなります。本人の 合計所得 1000万円 (年収 1195万円) 以下、12月31日時点で 入籍済み、生計を 一に している、青色事業専従者で ない、等が 主な 適用要件です。

配偶者控除の 計算方法 — 段階的な 控除額

配偶者控除 (配偶者 年収 103万円 以下)

  • 本人 年収 1095万 以下: 控除 38万円 (所得税)、住民税 33万円
  • 1095〜1145万: 26万円 / 1145〜1195万: 13万円 / 1195万超: 0円
  • 70歳以上の 配偶者 (老人控除): 48万円 (本人 1095万以下)

配偶者特別控除 (配偶者 年収 103〜201万)

  • 段階的に 控除額が 減少 (38万 → 36 → 31 → 26… → 3万)
  • 本人 年収 1195万 超 は 全額 不可
  • 150万円 を 超えると 急速に 減るので 「150万の 壁」

配偶者控除の よくある質問 (FAQ)

103万の 壁・150万の 壁 とは?
103万: 配偶者 自身に 所得税 発生 + 配偶者控除 → 特別控除 へ 切替の 境目。150万: 特別控除 が 段階的に 減り始める 境目。所得税 だけ 見れば 103〜150万 は 控除額 不変。
106万・130万の 壁 は?
これは 社会保険の 壁 (本ツールは 税金の 壁のみ)。106万: 大企業 パートで 社保加入。130万: 配偶者の 扶養から 外れ 国保・国民年金。当サイトの 年収の壁 判定 で 確認可能。
共働きで お互い 高収入なら?
本人 年収 1195万 超 だと 配偶者控除 ゼロ。共働き で それぞれ 高収入の 場合は 控除 メリット なし。
節税額は どうやって 出している?
控除額 × (所得税率 + 住民税10%)。本人の 年収から 所得税率 を 推計しています。
