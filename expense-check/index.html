🧾 個人事業主・フリーランス 経費にできる? 判定

確定申告で 「これって 経費?」 と 迷う 70項目を 一覧 + キーワード検索 で 即時判定。家事按分 (自宅兼事務所の 家賃・光熱費 等) の 目安%も 表示。

国税庁の 通達 + 一般的な 税理士 見解 ベース。最終 判断は 税務署 / 税理士 に 確認を。

該当する 項目が ありません。

🔒 すべて ブラウザ内 処理。サーバーには 何も 送信されません。

💡 経費 計上の 基本ルール

  1. 業務との 関連性 — 売上に 直接 or 間接的に 結びつくこと。私的な 支出 は NG。
  2. 領収書・レシート の 保管 — 7年間 保管義務 (帳簿は 10年)。電子帳簿保存法 対応。
  3. 家事按分 — 自宅兼事務所 等 業務 + 私用 混在の 場合、業務 使用率で 案分 (面積比・時間比・回数比)。
  4. 10万円以上の 資産 — 一括 経費 NG。減価償却 (年数で 分割)。
  5. 勘定科目 — 適切な 科目に 分類 (消耗品費・通信費・地代家賃・交際費 等)。

よくある質問

レシート しか ない 場合 でも 経費 にできる?
OK です。但し 5万円 超 (税抜) の レシートは 取引先名・支払先・金額が 必要。手書き 領収書 推奨。
家族との 食事 は 経費?
原則 NG。ただし 家族が 業務 委託 契約で 取引している 場合 (青色 専従者給与 等) は 「打ち合わせ」 として OK の 可能性 あり。
クレジットカード で 支払った 場合 は?
クレカ 明細 + 利用 店舗の レシート の セット保管 で OK。事業用 カードを 分けると 仕訳が 楽。
白色 申告 と 青色 申告 で 経費 ルール は 違う?
経費の 基本ルール は 同じ。青色は さらに 「青色専従者給与」 「30万円未満 一括 経費」 等の 特典あり。
税務調査で 否認される リスクは?
「業務 関連性が 説明できない」 「家事按分の 根拠が ない」 が 典型。証憑 + メモ で 業務 関連性を 残しておく。
