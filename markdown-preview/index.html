📝 Markdown ライブプレビュー

Markdownをリアルタイムで HTML プレビュー化。GitHub Flavored Markdown (GFM) 対応：テーブル・チェックリスト・コードハイライト・取り消し線。ブラウザ内処理でドキュメントはサーバーに送信されません。

📥 Markdown0 文字 / 0 行

👀 プレビュー--

🔒 すべてブラウザ内処理。書いている内容はサーバーに送信されません。

📖 対応している記法

  • 見出し: # H1 から ###### H6
  • 強調: **太字** *斜体* ~~取り消し~~
  • リスト: 番号付き 1. ...、箇条書き - ...、ネスト対応
  • タスクリスト: - [ ] 未完了、- [x] 完了 (GFM)
  • テーブル: パイプ | 区切り、:---: でアライン指定 (GFM)
  • リンク・画像: [テキスト](URL) ![alt](URL)
  • 引用: > ...
  • コード: `inline` および ``` で囲んだコードブロック (言語指定可)
  • 水平線: --- または ***
  • 改行: 行末に半角スペース2個 または 空行

❓ よくある質問

HTMLとして書き出せる？
はい。「📋 HTMLコピー」または「💾 .html保存」で取得可能。CSS は別途必要なら自分で追加。
記事の下書きに使える？
もちろん。書いた内容はブラウザのローカルストレージに自動保存されるので、ページを閉じても次回続きから書けます。
Mermaid (チャート図) 対応？
本ツールは GFM 標準範囲で、Mermaid は未対応。将来対応予定。
シンタックスハイライト対応言語は？
軽量実装のため、本ツールは基本のコードブロック表示のみ。ハイライトには Prism.js などを HTML 書き出し後に追加してください。
機密文書も書ける？
はい。ブラウザ内処理 + ローカル保存。書いた内容はサーバーに送信されません。
