🏠 住宅ローン控除 計算 シミュレーター

マイホーム 購入で 大きな 節税 メリットの ある 「住宅ローン控除」。借入額・所得・住宅 種別 から、13年間 (or 10年) の 累計 還付額 を 即時 算出。年末残高 × 0.7% の 仕組みに 準拠。

住宅ローン控除 (住宅借入金等 特別控除) は、マイホームを 住宅ローンで 購入・新築・増改築した 場合に、年末ローン残高 × 0.7% を 所得税 (引ききれない 分は 住民税 9.75万円まで) から 直接 差し引ける 国の 制度です。新築・長期優良住宅 / ZEH水準 / 省エネ基準適合 / 一般住宅、中古住宅 / 認定中古 などの 区分により 借入上限が 2000万〜5000万円、控除期間も 10年 / 13年 と 変動し、子育て世帯・若者夫婦 (どちらか 40歳未満) には 令和7年も 上乗せ控除が 継続します。本ツールでは 借入額・期間・金利・住宅区分・年収・世帯属性を 入力するだけで、元利均等返済を 前提に 各年の 年末残高を 自動算出し、13年間の 累計還付額・年別 控除額 を ブラウザ内で 即時シミュレーション。家計に 大きな インパクトを 与える 住宅ローン控除を 物件選定 前に 把握できます。

令和7年 (2025年) 入居 ベース。長期優良 / ZEH / 省エネ / 一般 / 中古 対応。

住宅ローン控除 とは — 制度の 概要

住宅ローン を 借りて マイホームを 取得した 人が、年末ローン残高 × 0.7% を 13年 (新築) or 10年 (中古) にわたって、所得税・住民税から 直接 差し引ける 制度。会社員は 初年度のみ 確定申告、2年目以降は 年末調整で OK。所得控除 (医療費・iDeCo 等) ではなく 「税額控除」 のため、節税 効果は 抜群で 累計 数百万円 単位の 還付に なる ケースも 珍しく ありません。

住宅ローン控除の 計算方法 — 要件と 上限

主な 要件

  • ローン期間 10年以上
  • 床面積 50㎡以上 (合計所得 1000万以下なら 40㎡以上 OK)
  • 入居 から 6ヶ月以内 に 住んでいる
  • 合計所得 2000万円 以下
  • 令和6年 以降の 新築: 省エネ基準 適合 が 必須

住宅ローン控除の よくある質問 (FAQ)

所得税で 引ききれない 分は?
住民税からも 控除可能 (上限 9.75万円/年)。それでも 残った 分は 還付されません。
初年度の 手続きは?
初年度のみ 確定申告 必須。2年目以降は 年末調整で OK (会社員)。「住宅ローン年末残高証明書」 + 「住宅借入金等特別控除証明書」 を 会社へ。
ペアローン だと どうなる?
夫婦 それぞれが 単独で 控除を 受けられる ため、合計 控除額が 大きく なります。共働き なら 検討価値あり。
繰上げ返済すると 損?
残高 が 減ると 控除額も 減ります。金利 0.5% で 控除 0.7% なら、控除期間中は 繰上げ せず 投資 等に 回す ほうが 有利な ケースが 多い (要試算)。
