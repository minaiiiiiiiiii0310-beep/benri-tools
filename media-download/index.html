📥 動画・画像ダウンロード補助

URLを貼るだけで、ファイル種別を自動判定:

  • 直リン (.jpg/.mp4/.pdf 等) → ブラウザ内で直接ダウンロード
  • Webページ → 中の画像・動画 URL を抽出してリスト化
  • YouTube/Vimeo/Twitter/TikTok → yt-dlp コマンドを生成（ご自身のPCで実行）
⚠️ 著作権・利用規約について必ずご確認ください
第三者の作品（YouTube動画・他サイトの画像など）を無断でダウンロード・転載することは、著作権侵害や利用規約違反になる場合があります。
本ツールはあなた自身がアップロードしたコンテンツのバックアップ・許諾済みのもの・Public Domain/Creative Commons 素材等の合法的な用途を想定しています。違法ダウンロードによる損害・処罰について本ツール提供者は責任を負いません。

✅ できること / ❌ できないこと

✅ できること
  • 直リン画像 (.jpg/.png/.webp): クロスオリジンでも CORS が緩ければ DL 可能
  • 直リン動画 (.mp4/.webm): 上と同じ
  • PDF 直リン: ほぼ全環境で OK
  • あなた自身のサイト: 自分のサーバー上のファイル
  • YouTube/Vimeo/Twitter/TikTok: yt-dlp コマンドを生成（自PC実行）
  • WebページのHTML: CORS-OK の場合は <img> URL を抽出
❌ できないこと（技術的制約）
  • ほとんどのニュースサイト・SNSの画像は CORS で fetch ブロック → 「右クリック→保存」案内に切り替え
  • YouTube/Netflix 等の DRM 保護動画は yt-dlp でも DL 不可
  • 会員制サイトのログイン後ページ → ログインセッションがブラウザにあれば自分で右クリック保存
  • Instagram/Facebook のプライベート投稿
❌ できないこと（法的制約）
  • YouTube は利用規約でダウンロードを禁止 → yt-dlp 使用は自己責任
  • 有料コンテンツ・著作権で保護されたコンテンツの無断DL は違法
  • 日本の著作権法では、違法アップロードと知りつつDLすると刑事罰の対象（2021年改正）

📦 yt-dlp のインストール方法

yt-dlp は YouTube/Vimeo/Twitter/TikTok など 1,700+ サイト対応のオープンソース動画ダウンローダー。

1Windows: winget install yt-dlp.yt-dlp または yt-dlp.exe をダウンロード
2Mac: brew install yt-dlp
3Linux: sudo pip install yt-dlp
4ffmpeg も推奨（音声・動画結合用）: Windows winget install ffmpeg / Mac brew install ffmpeg

本ツールが生成したコマンドをターミナル/コマンドプロンプトに貼り付けて実行してください。

🌐 なぜブラウザだけで全部 DL できないのか

ブラウザのセキュリティ機構「CORS (Cross-Origin Resource Sharing)」により、他サイトのリソースを fetch するには、そのサイトが許可ヘッダを返す必要があります。

  • CORS 許可済み: Wikipedia, Unsplash, Pexels, GitHub raw 等 → fetch + ZIP 可能
  • CORS なし: ほとんどのニュースサイト・SNS → fetch 不可。「右クリック→保存」が必要
  • 解決策: 「画像を新規タブで開く → 右クリック → 名前を付けて保存」が最も確実

YouTube は 暗号化された動画ストリームを配信するため、URLを直接 fetch しても再生できない形式。yt-dlp は復号・結合まで行うため必要。

❓ よくある質問

このツールはなぜ YouTube 動画を直接ダウンロードできない？
YouTube は ToS でダウンロードを禁止しています。また技術的にも、ブラウザから直接 YouTube の動画ストリームを取得することは CORS と暗号化によりできません。yt-dlp 等を自分の PC で使うことになります。
yt-dlp は合法？
yt-dlp 自体は合法 (MITライセンスのOSS)。ただし、それで取得したコンテンツの利用は著作権法・各サービスのToSに従う必要があります。自分のチャンネル・パブリックドメイン・Creative Commons コンテンツは問題なし。
違法ダウンロードの罰則は？
2021年改正著作権法により、違法アップロードと知りつつDLすると2年以下の懲役 or 200万円以下の罰金（私的使用でも）。スクショや写り込みは対象外。
自分のサイトのバックアップに使える？
はい。自分が運営するサイトの画像・動画のバックアップなら全く問題なし。むしろ推奨。
Instagram の自分の投稿を保存したい
Instagram の公式機能「アーカイブ」を使うのが最も簡単。投稿一覧画面で「・・・」→「保存」も可能。yt-dlp も対応ですがログインCookieが必要で難易度高め。
X (Twitter) の動画を保存したい
自分のツイートの動画 → アナリティクスからDL可能。他人の動画は yt-dlp で URL貼って実行（自己責任）。
