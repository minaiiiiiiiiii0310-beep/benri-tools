🔄 CSV ⇄ JSON ⇄ TSV ⇄ Markdown 変換

CSV/TSV/JSON/Markdownテーブルを相互変換。Excel から直接貼り付け OK。引用符・カンマ・改行を含むセルも正しく処理。すべてブラウザ内処理でサーバーには送信されません。

📑 使い方
  • 左に元データを貼り付け（Excel/Googleシートからコピペも可）
  • 形式を選ぶ（自動検出も可）→ 右に変換結果が即時表示
  • 右上の「コピー」「ダウンロード」で結果を取得

  • ネストされたJSON のフラット化: 階層のあるJSONを2次元テーブルに展開

❓ よくある質問

データは送信されますか？
いいえ、すべてブラウザ内処理（JavaScript）です。機密データや個人情報を含むCSVも安心して使えます。
カンマや改行を含むセルはどう処理されますか？
RFC 4180準拠で引用符付きとして処理。"a, b" のようなセルも正しくパース・出力します。
JSON配列がネスト構造のとき？
第1レベルでフラット化します。深い階層が必要な場合はカスタム処理が必要です。
大量データは大丈夫？
10万行までサクサク。それ以上は処理に数秒かかりますが動作します。
Excel の貼り付けがうまく行かない
セルをコピーすると自動的にTSV (タブ区切り) になります。「自動検出」または「TSV」を選んでください。
