使い方のコツ

📚 結合
複数のPDFを 1つに まとめる。ファイル一覧の × ボタンで 除外、ドラッグで 並び替え 可能。会議資料・領収書 まとめに。
✂️ 分割
大きなPDFを ページ範囲で 切り出し。範囲は カンマ区切りで 複数指定 可能 (例: 1-3,5,7-9 → 3ファイルに 分割)。
🖼 PDF→画像
PDFの各ページを 個別の 画像ファイルに。ZIP で 一括 ダウンロード。スライド・図面の SNS共有 等に。
📥 画像→PDF
スマホで 撮った 書類写真などを 1つのPDFに まとめ。提出書類・スキャン代わりに。

よくある質問

ファイルは どこに 送られますか?
どこにも 送られません。PDFの 処理は すべて あなたの ブラウザ内で 完結。サイトの サーバーには 一切 アップロード されません。
ファイル サイズの 制限は?
ブラウザの メモリ次第。一般的に 100MB までは 問題なく 動作します。それ以上の 巨大ファイルは メモリ不足で 失敗する 可能性 (PCのメモリに 依存)。
パスワード付き PDF は?
本ツールは パスワード保護された PDF の 処理は サポートしていません。事前に Adobe 等で ロック解除 してから ご利用 ください。
結合した PDF の 順番を 変えたい
ファイル一覧で 各項目を ドラッグして 順序変更 できます。
