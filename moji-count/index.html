0

0

🔒 入力した文章はあなたのブラウザ内だけで処理され、サーバーには送信されません。

よくある質問

全角・半角はどう数えますか？
日本語などは全角、英数字・記号は半角として個別にカウントします。
原稿用紙の枚数は？
空白を除いた文字数を400字で割って算出します（端数は切り上げ）。
入力した文章は保存されますか？
いいえ。すべてブラウザ内で処理され、どこにも送信・保存されません。
