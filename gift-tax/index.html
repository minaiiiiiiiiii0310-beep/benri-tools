🎁 贈与税 計算 シミュレーター

親・祖父母 等から お金や 不動産を もらった ときに かかる 贈与税。暦年贈与 (110万円 基礎控除) と 相続時精算 課税 (2500万円 控除) を 自動 比較。

贈与税 は、個人から 個人へ お金・不動産・株式 等の 財産を 無償で 譲り渡した ときに 受贈者 (もらう側) に 課される 国税です。相続税の 補完税 として 高い 税率 (最大 55%) が 設定されている 一方で、暦年贈与 (年間 110万円 非課税) や 相続時精算 課税 (累計 2500万円 控除 + 令和6年改正で 毎年 110万円 基礎控除 追加)、住宅取得資金 贈与 (最大 1000万円)、教育資金 一括贈与 (1500万円)、結婚・子育て資金 (1000万円) など 多くの 特例も 存在します。本ツールは 贈与額・贈与者と 受贈者の 関係 (直系尊属の 特例税率 / 一般税率)・受贈者の 年齢を 入力するだけで、令和6年 改正後の 国税庁 速算表を 用いて 贈与税額を 即時算出し、暦年贈与と 相続時精算 課税の 有利不利 を 自動比較します。

直系尊属 → 18歳以上 の 子・孫 への 贈与 は 「特例税率」 で さらに 安く。

贈与税 とは — 課税対象と 非課税枠

個人から 個人への 財産移転に かかる 国税。受贈者 (もらう人) が 申告納税します。1月〜12月の 1年間の 贈与額が 110万円を 超えたら 翌年 2/1〜3/15に 確定申告が 必要。配偶者の 居住用 不動産 贈与 (2000万円控除)・教育資金 一括贈与・結婚子育て資金 など 特例も 多く、活用すれば 大きく 節税できます。

贈与税の 計算方法 — 2つの 課税方式

① 暦年贈与 (毎年 110万円 まで 非課税)

  • 1月〜12月の 贈与額 - 110万円 = 課税対象
  • 速算表で 税額算出
  • 令和6年 改正: 相続前 7年以内 の 贈与は 相続財産に 加算

② 相続時精算 課税 (累計 2500万 まで 非課税)

  • 60歳以上の 親・祖父母 → 18歳以上の 子・孫
  • 累計 2500万まで 贈与税ゼロ、超過分は 一律 20%
  • 相続時に 全額を 相続財産に 加算
  • 令和6年改正: 毎年 110万円 の 基礎控除 追加 (持ち戻し なし)

贈与税の よくある質問 (FAQ)

110万円 以下 なら 申告 不要?
はい。基礎控除 内なら 贈与税ゼロ・申告不要。ただし 名義預金 (親が 通帳・印鑑を 管理) は 贈与と 認められない ことが あるので 注意。
毎年 110万 ずつ 贈与すれば 安全?
定期 (連年) 贈与 と みなされる リスクあり。毎年 契約書 + 振込 + 金額・時期を 変える が 推奨。令和6年改正で 相続前 7年は 加算対象に。
住宅 取得 資金の 特例?
父母・祖父母から 住宅取得 のために もらう お金は 最大 1000万円 (省エネ住宅) まで 非課税。通常の 110万 と 併用可能。
申告 期限は?
贈与を 受けた 翌年の 2月1日〜3月15日。期限後は 加算税・延滞税 が 発生。
