カラーコード変換（HEX ⇔ RGB）

HEXコード（#RRGGBB）とRGB値を相互に変換します。hex→rgb / rgb→hex どちらもOK。カラーピッカーで選んでも変換できます。CSSやデザインにそのまま使えます。

使い方

HEX → RGB に変換
HEX欄に「#2b7de9」のように入力すると、RGBとプレビューが更新されます。「#fff」のような3桁表記もOKです。
RGB → HEX に変換
RGB欄に「43,125,233」または「rgb(43,125,233)」と入力すると、HEXが更新されます。
色を選んで変換
カラーピッカーで色を選ぶと、HEX・RGBの両方が自動で表示されます。

HEXとRGBの違い

同じ色でも、表記の方法が2通りあります。用途に合わせて使い分けます。

  • HEX（16進数）：#RRGGBB の形式。HTML/CSSでよく使われます（例：#2b7de9）。
  • RGB（10進数）：赤・緑・青をそれぞれ0〜255で表します（例：rgb(43, 125, 233)）。透明度を加えた rgba() もあります。

よく使う色のカラーコード一覧

白 / 黒
白 = #ffffff（rgb(255,255,255)）／黒 = #000000（rgb(0,0,0)）
赤 / 緑 / 青
赤 = #ff0000（rgb(255,0,0)）／緑 = #008000（rgb(0,128,0)）／青 = #0000ff（rgb(0,0,255)）
グレー系
明るいグレー = #f3f4f6 ／ グレー = #9ca3af ／ 濃いグレー = #374151
よく使う差し色
ブルー = #2563eb ／ グリーン = #16a34a ／ オレンジ = #ea580c ／ レッド = #dc2626

よくある質問

3桁の短縮HEX（#fffなど）も変換できますか？
はい。3桁表記は自動で6桁に展開して変換します。
入力した色はどこかに送信されますか？
いいえ。すべてブラウザ内で処理され、外部に送信されません。安心してお使いください。
