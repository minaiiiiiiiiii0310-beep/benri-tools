💊 医療費控除 計算 シミュレーター

年間に 支払った 医療費が 多い人は、確定申告で 所得税・住民税 が 戻ってきます。世帯合算 OK、薬局の OTC 薬も 対象。10万円ルール + セルフメディケーション 税制 を 自動で 有利な方に 判定。

医療費控除 は、1年間 (1月〜12月) に 支払った 医療費が 一定額を 超えた 場合、その 超過分を 所得から 控除できる 制度です。会社員でも 確定申告すれば、源泉徴収された 所得税の 一部が 還付され、翌年の 住民税も 安く なります。控除対象は 病院の 診察・治療費、処方薬、入院費、出産費 (帝王切開を 含む)、歯科治療 (自費含む)、視力回復 レーシック、通院 公共交通費、介護保険 サービスの 自己負担分など 幅広く、家族 (同一生計) の 分を 合算できる ため、共働き 世帯では 所得税率の 高い 人で 申告 するのが 節税の 鉄則。スイッチOTC薬 中心の 人は セルフメディケーション 税制 (88,000円 上限) も 選択可能で、本ツールは どちらが 有利か 自動 判定し、還付見込額を 即時 表示します。

給与所得 で 控除額 → 還付額 を 一気に 試算。

1. あなた・家族の 医療費

通院・入院・薬代・出産費・治療目的の マッサージ・歯科自費治療・通院 交通費 など。家族分も 合算 OK。

対象成分の マークが ある OTC薬 (ロキソニン・ガスター・アレジオン 等)。健康診断 受診者のみ 利用可。

2. 申告者の 所得

🔒 すべて ブラウザ内 処理。サーバーには 何も 送信されません。

医療費控除 とは — 対象者・対象範囲

1月〜12月の 1年間に 支払った 医療費が 10万円 (or 所得の5%) を 超えた 部分 を、所得から 控除できる 制度。確定申告 で 所得税・翌年の 住民税が 戻ってきます。会社員・公務員・個人事業主・年金受給者 など 確定申告できる 人 が 対象で、扶養家族の 医療費も 申告者に 合算可能。出産・入院・歯列矯正 (噛み合わせ治療)・通院 交通費・あんま指圧・はり・きゅう・治療目的の マッサージなど 想像以上に 多くの 費用が 対象に なります。

医療費控除の 計算方法 — 還付額の 出し方

計算式

  • 医療費控除額 = 年間医療費 - 保険給付 - min(10万円, 所得×5%)
  • 上限 200万円
  • 還付額 = 控除額 × (所得税率 + 住民税10%)

セルフメディケーション 税制 (特例)

  • スイッチOTC薬 等の 購入額 - 12,000円 = 控除額 (上限 88,000円)
  • 健康診断・予防接種 を 受けた 人のみ 利用可
  • 通常の 医療費控除 と 選択制 (両方は不可)

医療費控除の よくある質問 (FAQ)

家族の 分も 合算できる?
同一生計の 家族 (配偶者・子・親) なら 合算 OK。最も 所得税率の 高い人で 申告 すると 還付額が 最大化します。
領収書は 必要?
2017年以降 は 「医療費控除の 明細書」 を 提出すれば 領収書 不要。ただし 5年間 自宅保管 義務あり。マイナポータルから e-Tax で 医療費 自動入力 も 可能。
通院 交通費は 含まれる?
公共交通機関 (電車・バス) の 通院 交通費 は OK。自家用車の ガソリン代・駐車場代 は NG。タクシーは 急病・歩行困難 等 やむを得ない 場合のみ。
美容整形・人間ドックは?
原則 対象外。ただし 病気が 見つかった 人間ドックは その後の治療費と セットで 対象に。視力矯正 (眼鏡・コンタクト) は 治療目的のみ OK。
