🏥 高額療養費 計算 シミュレーター

入院・手術で 1ヶ月の 医療費 自己負担が 高額に なったとき、健康保険から 戻ってくる 「高額療養費」 の 自己負担 上限額 を 即時 計算。月収・年齢で 自動 区分判定。

「高額療養費 制度」 は、健康保険・国民健康保険・後期高齢者医療制度の 加入者が、1ヶ月の 医療費 自己負担額が 自己負担限度額 を 超えた 場合に、超えた 分が 払い戻される 仕組みです。入院・手術・がん治療・人工透析・出産 (帝王切開) など 高額な 医療を 受ける 前に、自分の 月収・年齢区分から 限度額 を 把握しておけば、家計の 不安を 減らせます。本ツールは 標準報酬月額・年齢・医療費総額・多数回該当の 有無を 入力するだけで、令和7年度 基準の 5区分 (ア〜オ) または 70歳以上区分を 自動判定し、自己負担上限・払い戻し額・実質負担額を ブラウザ内で 即時算出。マイナンバーや 個人情報の 入力は 一切 不要、計算結果は 外部サーバーに 送信されません。

マイナンバー・個人情報 不要。区分が わかれば 自己負担 上限が すぐ 確定。

1. 年齢・収入

給与明細の 健康保険料 ÷ 4.92% で 逆算可能。国保の方は 年収 ÷ 12。

2. 医療費

3割負担前の 総額。窓口請求額 ÷ 0.3 で 推計可能 (3割の場合)。

🔒 すべて ブラウザ内 処理。サーバーには 何も 送信されません。

高額療養費 制度とは — 仕組みと 対象者

1ヶ月 (月初〜月末) の 医療費 自己負担 が 一定額を 超えると、超えた分が 健康保険から 払い戻される 制度。申請しないと もらえない ので、入院・手術が 決まったら 必ず 「限度額適用認定証」 を 事前に 取得しましょう。対象は 会社員 (協会けんぽ・健保組合)、自営業 (国民健康保険)、75歳以上 (後期高齢者医療制度) の すべての 公的医療保険 加入者。生命保険の 医療保険とは 別物で、申請すれば 全員が 自動的に 受けられる 国の セーフティネットです。

高額療養費の 計算方法 — 区分別 自己負担限度額

70歳未満の 5区分 (令和7年度)

  • 区分ア: 標準報酬月額 83万円以上 → 252,600円 + (医療費-842,000) × 1%
  • 区分イ: 53〜79万円 → 167,400円 + (医療費-558,000) × 1%
  • 区分ウ: 28〜50万円 → 80,100円 + (医療費-267,000) × 1%
  • 区分エ: 26万円以下 → 57,600円
  • 区分オ: 住民税 非課税 → 35,400円

高額療養費の よくある質問 (FAQ)

多数回該当 とは?
直近 12ヶ月で 3回 以上 高額療養費の 対象に なった月が ある場合、4回目以降の 上限額が さらに 引き下げられます (区分ウなら 80,100円 → 44,400円)。
限度額適用認定証 は どうやって もらう?
会社員 は 健保組合、国保 は 市役所に 申請。マイナ保険証 を 使えば 認定証なしで 自動 適用されます (限度額情報の 提供同意が必要)。
世帯合算 できる?
同じ 健康保険 の 家族の 同月 自己負担額が 21,000円 以上 ある場合、合算して 計算 可能 (70歳未満 の 場合)。
差額ベッド代 や 食事代 は?
対象外。先進医療・自由診療・個室差額・食事療養費 は 高額療養費の 計算に 含まれません。
