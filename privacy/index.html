🔒 プライバシーポリシー

1. ツール 入力データの 取扱い

本サイト 各ツールに 入力された 数値・テキスト・画像 等は すべて あなたの ブラウザ内 (クライアントサイド) で 処理され、サイト運営者を 含む 第三者の サーバーに 送信・保存される ことは ありません。

ただし、以下の ツールは 外部 API を 利用するため、その 入力テキストが 外部に 送られます:

  • QRコード 生成: api.qrserver.com に テキストを 送信して 画像を 生成しています。機密データには 使わないでください。

2. Cookie・アクセス解析

本サイトは サイト改善 目的で Google Analytics 4 を 利用しています。Google Analytics は Cookie を 使用して 匿名の トラフィック データを 収集します。IPアドレスは 匿名化されており、個人を 特定できる 情報は 含まれません。

Google Analytics の オプトアウトは Google公式 オプトアウト アドオン から 可能です。

3. 広告配信

本サイトは Google AdSense による 広告配信を 行っています。Google および その パートナー (第三者配信業者) は Cookie を 使用して、利用者が 本サイトや 他の サイトに アクセスした 際の 情報に 基づいて 広告を 配信します。

パーソナライズド 広告の オプトアウトは Google 広告設定 から 可能です。

4. 免責事項

本サイトの ツールが 出力する 結果は 「概算」 「目安」 として 提供されており、その 正確性・最新性・有用性 等を 保証する ものではありません。重要な 判断・契約・申請・医療判断 等は 必ず 公式情報 または 専門家に ご相談ください。本サイトの 利用に 起因する 損害について 運営者は 一切の 責任を 負いません。

5. 外部リンク

本サイトには 外部 サイトへの リンクが 含まれます。外部 サイトの 内容・プライバシー ポリシーについて 本サイト 運営者は 責任を 負いません。

6. 著作権

本サイトの コンテンツ (テキスト・コード) の 著作権は サイト 運営者に 帰属します。ツールの 計算式は 公開情報 (法令・公的統計・業界慣行) に 基づきます。

7. 改定

本ポリシーは 予告なく 改定される ことがあります。重要な 変更が ある 場合は サイト上で お知らせします。
