📁 ファイル名 一括リネーム

複数ファイルを一気にリネーム。正規表現・連番付与・日付追加・前後置換を組み合わせ可能。プレビューで安全に確認してから ZIP 出力。ファイルはアップロードされずブラウザ内で処理。

📑 こんなときに使う
  • iPhone写真の IMG_1234.JPG を 2024-06-19_001.jpg 形式に
  • 不要な (コピー) や _final_v2 を一括削除
  • 連番だけのファイルにタイトルを前置 (001.mp3 → BGM_001.mp3)
  • 大文字小文字の統一・拡張子変更
📁

ファイルをドラッグ＆ドロップ または クリック

🔒 ファイルはアップロードされません。すべてブラウザ内処理。リネーム後はZIPでダウンロード→解凍して元の場所に上書き。

📖 ルールの種類

  • 置換 (テキスト): 「(コピー)」→「」のように指定文字列を別の文字列に置換
  • 置換 (正規表現): IMG_(\\d+) → photo_$1 のようなパターン置換
  • 前に追加: ファイル名の先頭にテキストを付加 (例: 2024_)
  • 後ろに追加: 拡張子の直前にテキストを付加 (例: _v2)
  • 連番追加: 001 002 ... の連番を追加
  • 日付追加: 今日の日付 (YYYY-MM-DD) を追加
  • すべて新名: 名前を完全に置き換え + 連番

❓ よくある質問

ファイルはアップロードされる？
いいえ。ブラウザ内 (JavaScript File API + JSZip) で完結。ファイル本体はサーバーに送信されません。
フォルダごとリネームできる？
Web 標準制限でフォルダ構造は再現できません。ファイルだけ選択してZIPにまとめます。
同名ファイルが衝突したら？
プレビューで赤くハイライトされます。連番ルールで自動回避するか、手動で別ルールを追加してください。
大量ファイルでも大丈夫？
1,000ファイル程度までサクサク。それ以上は ZIP生成に時間がかかります。
Mac の HEIC 写真もリネームできる？
はい。ファイル種類問わずリネーム可能 (変換はできません。HEIC→JPG変換はその専用ツールを使用)。
